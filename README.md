## DESCRIÇÃO
Neste projeto, você terá a oportunidade de otimizar o Sistema Bancário previamente desenvolvido (Versão 1) com o uso de funções Python. O objetivo é aprimorar a estrutura e a eficiência do sistema, implementando as operações de depósito, saque e extrato em funções específicas. Você terá a chance de refatorar o código existente(versão 1), dividindo-o em funções reutilizáveis, facilitando a manutenção e o entendimento do sistema como um todo.
